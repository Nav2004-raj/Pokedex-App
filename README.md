# 🧭 Pokédex App

A simple and interactive Pokédex web application built using **HTML, CSS, and JavaScript**. This app allows users to explore Pokémon, search them by name or number, and view their abilities.

---

## 🚀 Features

* 🔍 Search Pokémon by **name or ID**
* 📋 Display list of all Pokémon on homepage
* ⚡ Click on a Pokémon to view detailed **abilities page**
* 📱 Responsive and user-friendly design
* 🔄 Smooth page navigation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Pokémon API (for fetching data)
